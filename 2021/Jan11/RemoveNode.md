# Remove Nth Node From End of List

Given a linked list, remove the n-th node from the end of list and return its head.

Examples:
```
Inputs:
list = 1->2->3->4->5
n=2
Output: 1->2->3->5


list = 1->2->3->4->5
n=1
Output: 1->2->3->4
```

Note: n will always be a valid integer over zero.
