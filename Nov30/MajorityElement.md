# Majority Element

Given an array of size `n`, find the majority element. The majority element is the element that appears more than `n/2` times.


Examples:
```
Input: [3,2,3]
Output: 3

Input: [2,2,1,1,1,2,2]
Output: 2

Input: [1, 2, 3, 4, 5]
Output: null
```

Source: [LeetCode](https://leetcode.com/problems/majority-element/)
