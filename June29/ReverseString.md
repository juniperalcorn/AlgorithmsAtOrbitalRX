# Reverse String

Given a string, reverse and return the string.

Example:
```
Input: "cat"
Output: "tac"

Inpupt: "Hello"
OutputL "olleH"

Input: ""
Output: ""
```

Case doesn't matter.

Challenge cases: don't use built in methods.
