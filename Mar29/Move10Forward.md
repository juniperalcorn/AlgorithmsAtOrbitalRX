# Move 10 Forward

Move every letter in the provided string forward 10 letters through the alphabet.

If it goes past 'z', start again at 'a'.

Input will be a string with length > 0.

Examples: 
```
Input: "testcase"
Output: "docdmkco"

Input: "codewars"
Output: "mynogkbc"

```
